# Libft – FR / EN README
## ■■ English Version
Libft is my first project at 42.
It is a custom C library that reimplements essential functions from the standard C library, along with
additional utility functions and a full linked list API.
### Features
- Reimplemented libc functions (memset, memcpy, atoi, strlcpy, etc.)
- String utilities (substr, split, strjoin, strtrim)
- Conversion (itoa)
- Function mapping (strmapi, striteri)
- File descriptor output (putchar_fd, putnbr_fd, etc.)
- Bonus: Full linked list API (lstnew, lstadd_front, lstclear, …)
### Build
make
make bonus
make clean
make fclean
make re
### Purpose
This project strengthened my fundamentals in C, memory management, pointer logic, and modular
programming.
